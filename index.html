What is Smartschool+?

Smartschool+ is a tool for Smartschool, giving you extra control with database acces, personalized profiles, grade calculations, and interface customization. It's designed so you can better monitor results and give more control over your Smartschool expierence.

What can you do with Smartschool+?

  • Customizable Top Bar

    Change the background color of the Smartschool interface and even enjoy a rainbow mode that cycles through colors.

  • Quick Shortcuts

    Enjoy quick shortcuts like a link to the "Planner" right on the top bar.

  • Profile Customization

    Personalize your profile by changing your username and profile picture with a simple key sequence ("change").

  • User Search

    Find other users with a simple key sequence ("find").

  • Settings Modal

    Adjust your preferences in the settings modal, including a slider to adjust the hue of the top bar and a checkbox to toggle between rainbow mode and slider mode.

  • Grid Feature

    Explore a grid feature, which seems to be a placeholder for a future feature. It checks for the existence of an element with the ID "show-grid" and alerts you if it exists.

  • Database Import

    Import data about all Smartschool users from a database with ease.
